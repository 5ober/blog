CSS 资源大全中文版

Blank 2020-04-18 css

我想很多程序员应该记得 GitHub 上有一个 Awesome：XXX 系列的资源整理。[awesome-css] 是 sotayamashita 发起维护的 CSS 资源列表，内容包括：CSS预处理器、框架、CSS结构、代码风格指南、命名习惯、播客、演讲视频、大网站的 CSS 开发经验等等。

Awesome 系列虽然挺全，但基本只对收录的资源做了极为简要的介绍，如果有更详细的中文介绍，对相应开发者的帮助会更大。这也是我们发起这个开源项目的初衷。

# 我们要做什么？

  • 基于 awesome-css 列表，我们将对其中的各个资源项进行编译整理。此外还将从其他来源补充好资源。
  • 整理后的内容，将收录在伯乐在线资源频道 (opens new window)。可参考已整理的内容： ：《BEM：前端命名方法论 (opens new window)》 ：《Sass：CSS预处理器 (opens new window)》 ：《YUI Compressor：JS/CSS压缩工具 (opens new window)》

# 如何参与本项目？

# 如何为列表贡献新资源？

欢迎大家为列表贡献高质量的新资源，提交PR时请参照以下要求：

  • 请确保推荐的资源自己使用过
  • 提交PR时请注明推荐理由

资源列表管理收到PR请求后，会定期（每周）在微博转发本周提交的PR列表，并在微博上面听取使用过这些资源的意见。确认通过后，会加入资源大全。

感谢您的贡献！

# 预处理器

更快地编译 CSS

  • GCSS：一个用GO语言编写的CSS预处理器。官网 (opens new window)
  • LESS：向下兼容CSS并为当前的CSS增加额外的功能。官网 (opens new window)
  • Myth (opens new window)：只用写纯CSS而不用担心浏览器加载缓慢。官网 (opens new window)
  • PCSS：一个用Python语言编写的CSS预处理器。官网 (opens new window)
  • PostCSS (opens new window)：通过JS插件来转换CSS。PostCSS (opens new window)
  • Sass (opens new window)：成熟、稳定且强力的专业CSS扩展语言。官网 (opens new window)
  • Stylus (opens new window)：用于nodejs的直观、强健、极具特色的CSS语言。官网 (opens new window)
  • YACP (opens new window)：另一种CSS预处理器。官网 (opens new window)

# 框架

  • 960 Grid System (opens new window)：简化了web开发工作流程。官网 (opens new window)
  • Blueprint (opens new window)：这个CSS框架为你提供易用的栅格系统、符合直觉的排版功能、有用的插件以及可打印的样式 官网 (opens new window)
  • Bootstrap (opens new window)：最流行的HTML、CSS、JS框架 官网 (opens new window)
  • inuit.css (opens new window) ：强力的、可扩展的、基于Sass的、采用BEM命名的面向对象CSS框架 官网 (opens new window)
  • Foundation (opens new window)：一个高级响应式前端框架。官网 (opens new window)
  • Material Design Lite (opens new window)：很好的用于制作Material Design风格网站的框架。官网 (opens new window)
  • Materialize (opens new window)：基于Material Design的现代响应式前端框架。官网 (opens new window)
  • Pure.css (opens new window)：一套可用于所有web项目的小型响应式CSS模块。官网 (opens new window)
  • Semantic UI (opens new window)：使用人性化html的强力框架。官网 (opens new window)
  • Skeleton (opens new window)：一个超简单的响应式模板。官网 (opens new window)
  • UIkit (opens new window)：适用于手机、平板以及电脑端的栅格系统。官网 (opens new window)

# CSS结构

  • RSCSS：CSS样式结构的合理标准。官网 (opens new window)
  • ITCSS：用于大型UI项目的稳定、可扩展、可控制的CSS架构。官网 (opens new window)

# CSS标准化

  • Normalize：一套提供较好的多浏览器默认样式一致性的CSS规范。官网 (opens new window)
  • Normalize OpenType：为Normalize.css添加了OpenType特性，如连字、字间距等等。官网 (opens new window)
  • Reset：一套CSS标准，将全部的HTML元素调整到一致的基准线。官网 (opens new window)
  • sanitize.css：一套可立即使用的，符合当今最优实践的CSS规范。官网 (opens new window)
