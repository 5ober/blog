Blank 2020-04-18 资源

我想很多程序员应该记得 GitHub 上有一个 Awesome - XXX 系列的资源整理。awesome-java (opens new window) 就是 akullpp 发起维护的 Java 资源列表，内容包括：构建工具、数据库、框架、模板、安全、代码分析、日志、第三方库、书籍、Java 站点等等。伯乐在线已经把 awesome-java 资源列表翻成中文后发布于 ImportNew (opens new window)。

Awesome 系列虽然挺全，但基本只对收录的资源做了极为简要的介绍，如果有更详细的中文介绍，对相应开发者的帮助会更大。这也是我们发起这个开源项目的初衷。

# 我们要做什么？

  • 基于 awesome-java 资源列表，我们将对各个资源项进行编译整理。
  • 整理后的内容，将收录在伯乐在线资源频道 (opens new window)。可参考已整理的内容：
    • 《OWNER：Java配置文件解决方案 (opens new window)》
    • 《Spring Boot：简化Spring应用初始搭建以及开发过程 (opens new window)》
    • 《SonarQube：开源的代码质量管理工具 (opens new window)》

# 如何参与本项目？

# 如何为列表贡献新资源？

欢迎大家为列表贡献高质量的新资源，提交PR时请参照以下要求：

  • 请确保推荐的资源自己使用过
  • 提交PR时请注明推荐理由

资源列表管理收到PR请求后，会定期（每周）在微博转发本周提交的PR列表，并在微博上面听取使用过这些资源的意见。确认通过后，会加入资源大全。

感谢您的贡献！

古董级工具

这些工具伴随着Java一起出现，在各自辉煌之后还在一直使用。

  • Apache Ant (opens new window)：基于XML的构建管理工具。官网 (opens new window)
  • cglib (opens new window)：字节码生成库。官网 (opens new window)
  • GlassFish (opens new window)：应用服务器，由Oracle赞助支持的Java EE参考实现。官网 (opens new window)
  • Hudson (opens new window)：持续集成服务器，目前仍在活跃开发。官网 (opens new window)
  • JavaServer Faces (opens new window)：Mojarra是JSF标准的一个开源实现，由Oracle开发。官网 (opens new window)
  • JavaServer Pages (opens new window)：支持自定义标签库的网站通用模板库。官网 (opens new window)
  • Liquibase (opens new window)：与具体数据库独立的追踪、管理和应用数据库Scheme变化的工具。官网 (opens new window)

构建工具

构建及应用依赖关系处理工具。

  • Apache Maven (opens new window)：Maven是一款声明式构建及依赖管理工具，采用约定优于配置方式进行管理。相对Apache Ant更推荐使用Maven，前者采用了过程式管理，维护相对困难。官网 (opens new window)
  • Bazel (opens new window)：来自Google的构建工具，可以快速、可靠地构建代码。官网 (opens new window)
  • Gradle (opens new window)：使用Groovy（非XML）进行增量构建，可以很好地与Maven依赖管理配合工作。官网 (opens new window)
  • Buck (opens new window)：Facebook构建工具。官网 (opens new window)

字节码操作

编程方式操作字节码的开发库。

  • ASM (opens new window)：通用底层字节码操作和分析开发库。官网 (opens new window)
  • Byte Buddy (opens new window)：使用流式API进一步简化字节码生成。官网 (opens new window)
  • Byteman (opens new window)：在运行时通过DSL（规则）操作字节码进行测试和故障排除。官网 (opens new window)
  • Javassist (opens new window)：一个简化字节码编辑尝试。官网 (opens new window)

集群管理

在集群内动态管理应用程序的框架。

  • Apache Aurora (opens new window)：Apache Aurora是一个Mesos框架，用于长时间运行服务和定时任务（cron job）。官网 (opens new window)
  • Singularity (opens new window)：Singularity是一个Mesos框架，方便部署和操作。它支持Web Service、后台运行、调度作业和一次性任务。官网 (opens new window)

代码分析

测量代码指标和质量工具。

  • Checkstyle (opens new window)：代码编写规范和标准静态分析工具。官网 (opens new window)
  • Error Prone (opens new window)：将常见编程错误作为运行时错误报告。官网 (opens new window)
  • FindBugs：通过字节码静态分析查找隐藏bug。官网 (opens new window)
  • jQAssistant (opens new window)：使用基于Neo4J查询语言进行代码静态分析。官网 (opens new window)
  • PMD (opens new window)：对源代码分析查找不良的编程习惯。官网 (opens new window)
  • SonarQube (opens new window)：通过插件集成其它分析组件，对过去一段时间内的数据进行统计。官网 (opens new window)

编译器生成工具

用来创建解析器、解释器或编译器的框架。

  • JavaCC (opens new window)：JavaCC是更加专门的轻量级工具，易于上手且支持语法超前预测。官网 (opens new window)
